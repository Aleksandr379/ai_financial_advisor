## Overview
**AI Financial Advisor** is a Python-based web application built with Streamlit.
It provides automated portfolio recommendations based on a user's risk profile,
investment horizon, and investment goals.

## How It Works
1. User selects investment horizon, risk tolerance, and goals
2. The system calculates a risk score
3. A risk profile is assigned (Conservative, Moderate, Aggressive)
4. A diversified portfolio is recommended
5. Results are visualized interactively
